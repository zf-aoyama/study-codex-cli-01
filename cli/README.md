# BTCマイニング簡易収益計算CLI

## 概要

BTCマイニングにおける簡易的な収益を計算する Python CLI ツールです。以下の入力値をもとに、1日の電気代、1日の売上、1日の利益、30日間の利益を計算します。

- ハッシュレート (TH/s)
- 消費電力 (W)
- 電気代 (円/kWh)
- BTC価格 (円/BTC)
- 想定採掘量 (BTC/日)

## セットアップ方法

このプロジェクトは `uv` で管理します。

```bash
uv sync
```

## 実行方法

```bash
uv run python main.py
```

## 実行例

```bash
uv sync
uv run python main.py
```

実行イメージ:

```text
BTCマイニング簡易収益計算ツール
以下の値を入力してください。
ハッシュレート (TH/s): 100
消費電力 (W): 3200
電気代 (円/kWh): 31
BTC価格 (円/BTC): 15000000
想定採掘量 (BTC/日): 0.00035

計算結果
ハッシュレート: 100.00 TH/s
1日の電気代: 2,380.80円
1日の売上: 5,250.00円
1日の利益: 2,869.20円
30日間の利益: 86,076.00円
```
