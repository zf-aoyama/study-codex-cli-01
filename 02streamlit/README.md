# BTCマイニング簡易収益計算ツール

BTCマイニングに必要な基本入力から、1日あたりと30日あたりの簡易収益を計算する Streamlit 製GUIツールです。

## ツール概要

- ハッシュレート、消費電力、電気代、BTC価格、想定採掘量を入力して収益を計算します。
- 1日の電気代、1日の売上、1日の利益、30日間の利益を日本円で表示します。
- 入力値が0以下の場合はエラーメッセージを表示し、画面が落ちないようにしています。

## セットアップ方法

`uv` を使って依存関係をインストールします。

```bash
uv sync
```

## 起動方法

以下のコマンドでアプリを起動します。

```bash
uv run streamlit run app.py
```

## 実行例

以下の値を入力した場合:

- ハッシュレート: 100 TH/s
- 消費電力: 3000 W
- 電気代: 31 円/kWh
- BTC価格: 15,000,000 円/BTC
- 想定採掘量: 0.0002 BTC/日

計算結果:

- 1日の電気代: 2,232 円
- 1日の売上: 3,000 円
- 1日の利益: 768 円
- 30日間の利益: 23,040 円
